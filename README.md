# MoviesDB
Movies Database Modular Architecture APP (Clean Arcitecture + MVVM)

## Features
- Fetching Movies and TvSeries
- Saving favorite content locally

## Frameworks and Libraries 
- Kingfisher
- RxSwift
- RxGesture
- Async/await concurrency for networking
- Core Data
- Combine
- Snapkit
- Uikit
- ProgressHud
